# Nassau Candy Shipping Route Efficiency Analysis

## Project Overview

This project analyzes factory-to-customer shipping efficiency for Nassau Candy Distributor using Python, Pandas, Matplotlib, and Streamlit.

The project focuses on identifying efficient and inefficient shipping routes, shipment delays, regional bottlenecks, and ship mode performance across multiple US regions.

---

## Objectives

* Analyze shipping lead times
* Identify fastest and slowest routes
* Perform regional bottleneck analysis
* Compare shipping modes
* Measure delay frequency
* Build an interactive analytics dashboard
